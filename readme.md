## Descripción

Sistema de gestión de datos de países desarrollado en Python.
Permite cargar, consultar, filtrar, ordenar y obtener estadísticas
a partir de un archivo CSV con información de países del mundo.

---

## Estructura del proyecto
gestion-paises-python/

├── main.py              # Punto de entrada y menú principal

├── datos.py             # Carga y gestión de datos

├── estadisticas.py      # Cálculos y métricas

├── filtros.py           # Filtrado por atributos

├── ordenamientos.py     # Ordenamiento de listas

├── validaciones.py      # Validación de entradas

└── data/

└── paises.csv       # Base de datos de países

---

## Cómo ejecutar

```bash
python main.py
```
